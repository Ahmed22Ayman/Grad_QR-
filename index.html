Emotional Intelligence (EI) Scrum Planner

Smarter Agile Management with Emotion Intelligence & Automated Task Assignment

🚀 Core Features

🤖 Automated Task Assignment

Uses AI to assign tasks based on developer performance, mood, and historical data — ensuring balanced workloads and optimized sprint flow.

📊 Smart Task Effort Estimation

Analyzes past sprint data to estimate how long tasks will take, helping improve sprint planning accuracy.

😊 Real-time Emotion Detection

Monitors developer emotions using computer vision to detect burnout risks and trigger adjustments.

🏆 Gamified Motivation System

Increases engagement through badges, points, leaderboards, and sprint rewards — turning productivity into a fun challenge.

🧠 AI Automation

Allows Scrum Masters to override AI suggestions and fine-tune sprint planning as the team evolves.

🧠 Tech Stack

  • Frontend: React.js, Axios, Tailwind CSS
  • Backend: Django, Django REST Framework, MySQL
  • AI & Analytics: TensorFlow, Computer Vision, Historical Sprint Data Analysis
